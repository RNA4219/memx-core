# データセット管理

## 目的
- 学習・検証に利用するデータセットの出所と識別情報を管理し、再現性と監査性を確保する。

## 更新責任者
- Data Steward（不在時は Repo Maintainer）。

## 更新頻度
- データセットの追加・更新・廃止が発生した都度。
